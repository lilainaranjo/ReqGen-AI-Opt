**Functional Requirements**

### Game Overview

1. **Game Name**: The Space Fractions game will be called "Space Fractions".
2. **Game Description**: The game is an interactive web-based game that helps sixth-grade students improve their fraction-solving skills.
3. **Gameplay**: Students will play the game, which will provide feedback on their game scores.
4. **Game Modes**: The game will have multiple game modes, including:
	* **Practice Mode**: Students will play a series of practice problems to improve their fraction-solving skills.
	* **Challenge Mode**: Students will play a series of challenge problems to test their skills.
	* **Quiz Mode**: Students will play a series of quizzes to test their knowledge of fractions.
5. **Gameplay Mechanics**:
	* Students will be presented with a series of problems, each with a specific fraction.
	* Students will have to solve the problem to progress to the next one.
	* Students will have a limited number of attempts to solve each problem.
6. **Feedback System**: The game will provide feedback on students' game scores, including:
	* **Correct Answers**: The number of correct answers students have.
	* **Time Taken**: The time students took to solve each problem.
	* **Accuracy**: The accuracy of students' answers.

### User Interface

1. **Game Menu**: The game will have a menu system that allows users to choose between the games.
2. **Game Selection**: The user will select the game they want to play from the menu.
3. **Game Mode Selection**: The user will select the game mode they want to play (Practice, Challenge, or Quiz).
4. **Gameplay Interface**: The game will have a graphical interface that displays the problems, answers, and feedback.
5. **User Input**: The user will input their answers to the problems.
