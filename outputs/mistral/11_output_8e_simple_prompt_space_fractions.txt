2. FR2: The system shall allow users to select a game from the menu. (Umbrella)
3. FR3: The system shall display the selected game. (Umbrella)
4. FR4: The system shall display a game interface with a space theme. (Space Fractions)
5. FR5: The system shall display a fraction problem on the screen. (Space Fractions)
6. FR6: The system shall allow users to select an answer for the displayed fraction problem. (Space Fractions)
7. FR7: The system shall provide feedback on the correctness of the user's answer. (Space Fractions)
8. FR8: The system shall display a new fraction problem after the user answers a question. (Space Fractions)
9. FR9: The system shall keep track of the user's score. (Space Fractions)
10. FR10: The system shall display the user's score at the end of the game. (Space Fractions)
11. FR11: The system shall provide feedback on the user's performance based on their score. (Space Fractions)
